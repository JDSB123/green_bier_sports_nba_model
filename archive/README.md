# Archive

This folder stores historical, version-controlled outputs for audits and ROI tracking.

Structure:
- slate_outputs/      Text outputs from `scripts/run_slate.py`
- analysis/           Slate analysis reports
- picks/              Betting card CSV exports
- predictions/        Full prediction CSV exports
- odds_snapshots/     Odds snapshots saved at prediction time

Notes:
- Files in this folder are committed to git for historical reference.
- Use unique timestamps in filenames to avoid overwrites.
